# Dice Fold Prototype

Базовый прототип браузерной игры, вдохновлённой формулой Dice & Fold: короткий roguelite-забег, враги-карты, d6-кубики и награды между комнатами. В проекте нет оригинальных ассетов или контента Dice & Fold — только самостоятельная механическая основа для дальнейшей разработки.

## Игровой цикл

1. Игрок входит в комнату и получает набор врагов-карт.
2. В начале хода бросается пул шестигранных кубиков.
3. Игрок выбирает кубик и назначает его на карту врага.
4. Значение кубика наносит урон; при 0 HP карта считается «сложенной».
5. Когда все кубики использованы, живые враги атакуют героя и начинается новый ход.
6. После очистки комнаты игрок выбирает одну реликвию и переходит дальше.
7. Победа наступает после складывания босса, поражение — при 0 HP героя.

## Архитектура

- `src/game/types.ts` — доменные типы состояния, кубиков, врагов, героя, реликвий и комнат.
- `src/game/content.ts` — стартовый контент: комнаты, враги, свойства врагов и реликвии.
- `src/game/engine.ts` — чистая логика забега: старт комнаты, броски, перебросы, назначение кубиков, ход врагов и награды.
- `src/ui/render.ts` — DOM-рендер, обработчики кликов и связь UI с игровым движком.
- `src/ui/styles.css` — визуальный слой карточного стола, врагов, кубиков, журнала и наград.
